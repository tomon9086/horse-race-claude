# 競馬予想スキル

出走表データを分析し、印（◎○▲△）と買い目を提案します。

## 実行手順

1. 会話内に出走表データがあればそれを使う。なければ `npx tsx src/fetch-entries.ts "$ARGUMENTS"` で取得する
2. 以下の分析軸でデータを分析する
3. 予想結果をフォーマットに従って出力する

## 分析軸

以下の観点から総合的に評価してください:

1. **血統分析**: 父系・母系の距離適性、馬場適性、コース適性
2. **実績分析**: 通算成績、クラス実績、重賞実績
3. **近走分析**: 直近4走の着順推移、上昇・下降トレンド、着差
4. **騎手分析**: 騎手の技量、当該コースでの実績、調教師との相性
5. **コース適性**: 距離適性（短縮・延長の影響）、芝/ダート適性、右回り/左回り
6. **展開予想**: 逃げ・先行・差し・追込の脚質分布、ハイペース/スローペースの予測
7. **斤量**: 負担重量の有利・不利、前走との増減

## 出力フォーマット

以下のフォーマットで出力してください:

```
## 予想: {レース名}

### 印
◎ {馬番} {馬名} — {一行理由}
○ {馬番} {馬名} — {一行理由}
▲ {馬番} {馬名} — {一行理由}
△ {馬番} {馬名} — {一行理由}
△ {馬番} {馬名} — {一行理由}

### 買い目

**馬連** (本線)
◎-○, ◎-▲

**ワイド** (押さえ)
◎-△, ○-▲

**三連複** (本線)
◎-○-▲, ◎-○-△

**三連単** (高配当狙い)
◎→○→▲, ◎→▲→○

### 各馬分析

{馬番} {馬名}
- 評価: A/B/C/D
- 血統: ...
- 近走: ...
- 展開: ...
- コメント: ...

### 注意事項
- 予想の根拠と信頼度を明示する
- 馬場状態（良/稍重/重/不良）が予想に与える影響に言及する
- 荒れそうな要素があれば指摘する
```

## 注意

- これは娯楽目的の予想です。投資判断の根拠にしないでください。
- 確定情報（枠順、騎手変更、馬場状態など）は当日まで変わる可能性があります。
