# オッズ取得スキル

netkeibaのrace_idを受け取り、単勝・複勝・馬連・ワイドのオッズを取得して出力します。

## 使い方

1. ユーザからrace_idを受け取る
2. race_idが12桁の数字であることを確認する
   - 一致しない場合は **コマンドを実行せず**、「race_idは12桁の数字で指定してください。（例: `202605021211`）」とユーザにエラーを返して終了する
3. `npx tsx src/fetch-netkeiba-odds.ts "$ARGUMENTS"` を実行する
4. 結果をそのまま出力する

## 出力内容

- 単勝オッズ（人気順）
- 複勝オッズ（人気順）
- 馬連オッズ（安い順 TOP30）
- ワイドオッズ（安い順 TOP30）

## 注意

- 前日・当日問わず取得可能（前日夜から発売開始）
- 直前はオッズが変動するため、発走直前に再取得すると精度が上がる
- 馬連・ワイドは前日発売開始前（前々日等）は空になることがある

## 引数

`$ARGUMENTS` にはnetkeibaのrace_idが入ります。指定されていない場合はユーザに確認してください。
