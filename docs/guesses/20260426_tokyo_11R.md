### ヘッジ戦略の意図

三連複は◎11軸の一頭流し（相手：1,3,5,7,12）。
◎11が大敗・競走中止した場合に三連複が全滅するリスクに備え、**11を含まない馬連4点**を追加する。

| 馬連 | 根拠 |
|------|------|
| 5-3 | ○-▲の最上位評価同士。11飛び時の最有力決着 |
| 5-7 | ラフターラインズ軸に差し脚質リアライズルミナス |
| 5-12 | ラフターラインズ軸にベンバトル×ディープの同系統 |
| 3-12 | サムシングスイート先行→スタニングレディ差しの展開カバー |

5番（○）を軸に3点すべてに絡めることで、11落馬時も○が来れば1点は必ず的中する構成。
馬連の購入額は三連複の1/2程度に抑えヘッジの役割に留める。

## 各馬分析

### 11番 ファムクラジューズ
- 評価: A
- 血統: ベンバトル（Galileo系欧州中長距離）× ディープインパクト。スタミナと末脚を兼備し東京中距離に最適
- 近走: 東京2000m未勝利1着(4人気)→東京2000mフリージア賞1着(2人気)。コースを掴んで連勝
- 展開適性: 差し〜追込。東京の長い直線で末脚を活かす形
- コメント: フリージア賞が7頭立ての少頭数は割引（knowledge.md）だが未勝利でも東京2000mを勝っており実績は本物。重賞初挑戦が唯一の不安

### 5番 ラフターラインズ
- 評価: A
- 血統: アルアイン（ロードカナロア×マルセリーナ）× キングカメハメハ。パワーと持続力で距離融通が利く
- 近走: 新馬2着→東京未勝利1着(1人気)→こうやまき賞3着(1人気)→きさらぎ賞G3 3着(4人気)。右肩上がりの安定成績
- 展開適性: 好位差し〜差し。どの展開にも対応可
- コメント: G3実績で格は最上位。D.レーンのトップ騎手効果（knowledge.md）加味。1800→2000mの延長も血統的に問題なし

### 3番 サムシングスイート
- 評価: A
- 血統: サートゥルナーリア（ロードカナロア×シーザリオ）× ヴィクトワールピサ。東京芝への適性が高い
- 近走: 牝未勝利5着→中山2000m2着(7人気)→東京1800m2着(1人気)→中山2000m牝未勝利1着(2人気)。成績上昇中
- 展開適性: 先行〜好位差し
- コメント: 2000mで2着→1着とステップアップ。内枠で流れに乗りやすい。酒井学騎手は格差し引き

### 7番 リアライズルミナス
- 評価: B
- 血統: シルバーステート（ディープ後継）× ルーラーシップ。スタミナ豊富で距離延長○
- 近走: 新馬5着→阪神1600m2着→京都1600m4着(1人気)→阪神1800m1着(3人気)
- 展開適性: 差し。後ろから追い込む形
- コメント: 前走初勝利後の重賞参戦でクラスの壁はあるが、松山弘平が東京コースで連れてくる可能性

### 12番 スタニングレディ
