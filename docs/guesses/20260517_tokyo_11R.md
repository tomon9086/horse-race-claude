## 各馬分析

### 12番 エンブロイダリー
- 評価: A
- 血統: アドマイヤマーズ（父）× クロフネ（母父）→ マイル適性◎ G1馬の血統
- 近走: 阪神牝馬S1着（1人気・1-1逃げ・上り33.5）→ 秋華賞1着（2人気・上り35.2）→ 桜花賞1着（3人気・上り34.0）
- 展開適性: 先行〜逃げ / 東京ではペースアップに注意だが前走で先行勝利確認済み
- コメント: 前走+14kgの体重増が気になる。当日の馬体重要確認。クイーンC（東京1600m G3）1着実績あり優勝フィルター該当。公式データ1番人気70%に乗れる最有力馬。ルメール鞍上で死角なし。

### 7番 クイーンズウォーク
- 評価: A-
- 血統: キズナ × Harlington → マイル〜中距離、差し
- 近走: 金鯱賞3着（1人気・中団・上り34.1）→ ヴィクトリアマイル2着（4人気・後方13-12・上り33.6）
- 展開適性: 後方待機〜差し / 東京1600mで後方から33.6の末脚使って2着
- コメント: 昨年の覇者候補。クイーンC1着・昨年VM2着と東京1600m重賞フィルター二重適用。前走金鯱賞3着（GII・1人気）でデータ「前走GII・4番人気以内」両方合致。今年も同舞台で変わり身は十分。体重542kgは18頭中最重量だが問題なし。

### 8番 カムニャック
- 評価: B+
- 血統: ブラックタイド × サクラバクシンオー（母父）→ 短〜中距離スピード
- 近走: 阪神牝馬S2着（4人気・4-3先行・上り33.2）→ 秋華賞16着（1人気・大敗）→ ローズS1着 → オークス1着（後方差し・上り33.8）
- 展開適性: 先行〜中団 / 秋華賞大敗は2000m距離不適、今回1600mは適距離
- コメント: 阪神牝馬Sで上り33.2の最速タイ末脚を記録してのハナ差2着。川田将雅が昨年のVM前哨戦（阪神牝馬S）2着馬に騎乗。秋華賞大敗はknowledge.mdの「距離不適→適条件回帰は積極評価」に該当。東京1600m重賞実績は（アルテミスS6着）でフィルター非該当だが実力は確か。

### 6番 ラヴァンダ
- 評価: B-
- 血統: シルバーステート × ベーカバド → マイル〜中距離
- 近走: 阪神牝馬S8着（3人気・4-5先行・上り33.7）→ 東京新聞杯2着（4人気・9-9差し・上り33.3）→ アイルランドT1着（4人気・上り32.4）
- 展開適性: 後方差し / 東京コースで上り32〜33秒台の切れ
- コメント: 東京新聞杯2着（東京1600m G3）で優勝フィルター該当。アイルランドTでは上り32.4の飛び抜けた末脚。ただし阪神牝馬S3人気8着は先行ポジションが仇。東京の後方差しならスタイルに合う。前走凡走の差し馬の変わり身に期待。

### 11番 ボンドガール
- 評価: C+
- 血統: ダイワメジャー × Tizway → マイル適性◎
- 近走: 中山牝馬S10着（5人気・後方・上り35.3）→ 小倉牝馬S2着（8人気）→ 関屋記念2着（2人気・上り32.8）
- 展開適性: 後方待機 / 切れ足は本物（上り32.8）
- コメント: 東京1600m重賞で過去2回3着以内（サウジアラビアRC2着・東京新聞杯2着）。関屋記念で上り32.8最速。ただし騎手が丹内祐次でG1では信頼度が下がる。前走中山牝馬S10着は状態面に疑問符。割引は必要だが東京コース適性と末脚で紐候補。

### 18番 チェルヴィニア
- 評価: C+
- 血統: ハービンジャー × キングカメハメハ → 中長距離指向だが上り勝負可
- 近走: 中山記念5着（5人気・後方・上り33.7）→ マイルCS10着（8人気・上り33.3）→ 毎日王冠7着（3人気）
- 展開適性: 後方〜中団差し / 切れは使えるが長距離向き馬体
- コメント: アルテミスS1着（東京1600m G3）で優勝フィルター該当。D.レーン騎乗で積極的な競馬。秋華賞・オークス勝ちの実績馬。マイルCSと桜花賞では凡走歴あり1600mへの適性にやや疑問も、東京コースは合う。
