# 2026年5月16日 新潟 11R 新潟大賞典

## レース概要
- 日時: 2026/05/16（土）
- 競馬場: 新潟（1回新潟5日）
- コース: 芝2000m・左回り・外回り
- 条件: GIII ハンデ戦（3歳以上）
- 出走頭数: 15頭

---

## 展開予想

**逃げ〜先行**: グランドカリナン（5番）が逃げ宣言級の2-2-2-2型。セキトバイースト（12番）は3-2-2-2の先行型だが2000mへの距離延長あり。シュガークン（15番）も2-2-2-1と先行する可能性。

**ペース想定**: グランドカリナンが単騎逃げを図り、ミドルペース（前半3F 36秒前後）になると予想。大きな先行争いにはならないが、外回り2000mで直線659mと長く、後方から差し届く展開になりやすい。

**脚質分布**（15頭）:
- 逃げ: グランドカリナン（5）
- 先行: セキトバイースト（12）、シュガークン（15）
- 中団: アンゴラブラック（4）、グランディア（3）、ヤマニンブークリエ（8）、ホールネス（1）
- 後方: ドゥラドーレス（6）、トーセンリョウ（7）、シンハナーダ（14）、シュトルーヴェ（13）

外回り直線が長く、4コーナー6〜10番手から末脚を使える馬が最も恵まれる。

## 各馬分析

### 6番 ドゥラドーレス
- 評価: A-
- 血統: ドゥラメンテ × ロカ（母父: ハービンジャー）→ 中距離2000m前後に適性。良馬場の芝向き
- 近走: AJCC2着(1人気)・上り34.5 → オールカマー2着(2人気)・上り34.4 → 七夕賞2着(1人気)・上り35.4 → 金鯱賞5着(2人気)・上り33.9
- 展開適性: 後方差し（9-8-8-7型）で新潟外回りの直線は最良の舞台
- コメント: 重賞で安定して2〜5着をキープ。金鯱賞でも上り33.9の切れを使っており状態は良好。最重量58kgは懸念だが、ルメールの騎乗技術でカバー可能。勝ち切れないパターンはあるが、今回はコース適性が高く逆転できると評価。

### 3番 グランディア
- 評価: B
- 血統: ハービンジャー × ディアデラノビア（母父: サンデーサイレンス）→ 左回り・外回りコース適性高い
- 近走: ディセンバーS2着(3人気) → 中山金杯3着(4人気)・上り34.0 → 大阪城S3着(2人気)・上り34.4
- 展開適性: 中団差し（6-6〜8-9-9型）。好位から差せるオールラウンダー
- コメント: 3戦連続で掲示板圏内の安定馬。中山金杯では上り34.0の末脚でハナ差3着に食い込んだ。ハービンジャー産駒は直線の長い新潟外回りに合いやすく、57kgのハンデでもこのメンバー構成なら十分に対応できる。

### 14番 シンハナーダ
- 評価: B
- 血統: レイデオロ × シンハディーパ（母父: ウォーエンブレム）→ 中距離2000m専門型
- 近走: 魚沼S1着(2人気)・上り32.7 → 中日新聞杯4着(2人気)・上り33.2 → 白富士S7着(5人気)・上り33.5
- 展開適性: 後方差し（14-12型）。直線に全てを賭けるタイプ
- コメント: 魚沼Sは新潟2000mでの上り32.7という驚異的な末脚。中日新聞杯4着でもGIII水準に対応済み。白富士Sは7着だったが上り33.5は使えており、バテたのではなくスローで差しが届かなかった可能性。56kgのハンデで新潟外回り直線は最大のチャンス。騎手の格（杉原誠人）がやや落ちる点が懸念。

### 7番 トーセンリョウ
- 評価: B
- 血統: ディープインパクト × カンビーナ（母父: Hawk Wing）
- 近走: エプソムC3着(10人気)・上り34.1・16-16-16 → ジューンS3着(3人気)・上り33.7 → オクトーバーS2着(5人気)・上り33.7 → 大阪城S7着(3人気)・-4kg
- 展開適性: 極後方差し（16-16型）。直線の長いコースが合う
- コメント: エプソムCで16番手から3着の実績が光る。上り33.7の末脚は安定しており、新潟外回りの659m直線は絶好条件。前走大阪城S7着は馬体重-4kgの状態面の問題と考えられ、その前の連続掲示板を基準に評価。56kgのハンデも標準的。

### 4番 アンゴラブラック
- 評価: B-
- 血統: キズナ × ブラックモリオン（母父: ルーラーシップ）→ 中距離2000m適性あり
- 近走: バーデンBC1着(1人気) → アイルランドT2着(6人気)・上り33.0！ → 中山金杯2着(1人気)・上り34.1 → 中山牝馬S13着(1人気)・稍重・-6kg
- 展開適性: 中団差し（3-5-4-3型）
- コメント: アイルランドTでの上り33.0は今回の出走馬の中でも最高レベルの末脚。中山金杯でも1人気2着と安定感あり。前走中山牝馬Sの1人気13着は稍重の馬場と馬体重-6kgの悪条件が重なった結果と考える。良馬場の新潟でキズナ産駒が本来の末脚を取り戻すか注目。

### 15番 シュガークン
- 評価: B-（休養リスクあり）
- 血統: ドゥラメンテ × シュガーハート（母父: サクラバクシンオー）→ 2400m青葉賞優勝馬、2000mも守備範囲
- 近走: 大寒桜賞1着(1人気) → 青葉賞1着(2人気)・上り33.9 → ダービー7着(8人気)・先行して失速
- 展開適性: 先行〜中団（2-2-2-1型）
- コメント: 2024年5月以来1年以上の長期休養明けが最大のリスク。ただし武豊騎乗で陣営の本気度は高いと推測。青葉賞では上り33.9の切れを使って1着。ドゥラメンテ産駒でパワーのある末脚を持つ。仕上がり次第では一発があるが、人気になるようなら買いづらい。

### 1番 ホールネス
- 評価: C
- 血統: Lope de Vega × Missunited（欧州系）
- 近走: エリザベス女王杯3着(2人気)・上り34.4 → 日経新春杯14着(5人気)・上り38.4 → 日経賞9着(14人気)・上り36.8
- コメント: エリザベス女王杯3着の実績と新潟牝馬S優勝の新潟実績は評価できるが、その後2連続大敗が続く。状態面の回復が必要。牝馬55kgのハンデは有利。

### 8番 ヤマニンブークリエ
- 評価: C
- 血統: キタサンブラック × ヤマニンプードレ → 距離2000〜2400m向き
- 近走: セントライト2着(8人気)・上り34.4 → 菊花賞16着(7人気) → 日経新春杯7着(5人気) → ネオムターフ5着
- コメント: 菊花賞で大敗したが3000mが合わなかったと考えられる。2000mへの距離短縮で適性が上がる可能性。横山典弘騎乗だが、近走の成績が安定しない。若馬のため成長可能性あり。

## 注意事項

- 馬場状態が稍重以上になると差し有利が更に強まり、◎ドゥラドーレス・▲シンハナーダに追い風。アンゴラブラックは稍重が合わないため△を下げる判断もあり。
- シュガークン（15番）は長期休養明けで仕上がり次第。パドック・オッズ動向を要確認。
- グランドカリナン（5番）がスローに落とすと先行馬が粘りやすくなり、セキトバイーストやシュガークンが台頭する可能性もある。
