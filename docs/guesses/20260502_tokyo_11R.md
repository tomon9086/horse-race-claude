# 2026年5月2日 東京 11R 京王杯スプリングカップ

## レース概要
- 日時: 2026/05/02
- 競馬場: 東京
- コース: 1400m・芝・左回り
- 条件: GII・3歳以上

## 展開予想
18頭立ての多頭数。逃げ候補はヤブサメ（外18番枠で武豊）もしくはウイントワイライト。先行勢はマイネルチケット・アサカラキング・ワイドラトゥールなど。
東京1400mは直線約525mと長く、先行しつつ直線で粘る「好位差し」スタイルが最もワークしやすい。スローよりも中〜平均ペースになりやすいが、18頭で外枠の先行馬には不利。逃げ馬が早めに動かされる展開になれば差し馬も台頭。

## 各馬分析

### 14番 マイネルチケット
- 評価: A
- 血統: ダノンバラード × マツリダゴッホ（マイル〜中距離に強い父系）
- 近走: 阪急杯9着（2人気/阪神1400m）→睦月S2着（1600m）→奥多摩S1着（東京1400m 1人気）→三年坂S2着（1600m）
- 展開適性: 好位差しが得意。東京コースで特に安定
- コメント: 東京1400m奥多摩S1着が直近で最も信頼できる東京コース実績。横山武史が引き続き騎乗。4歳で成長余地あり。阪急杯は2人気9着だったが阪神コースの相性問題の可能性。今回東京に戻って巻き返し期待。

### 5番 ワイドラトゥール
- 評価: A
- 血統: カリフォルニアクローム × アグネスタキオン（1200〜1400m適性に優れた配合）
- 近走: マイルCS17着（1600m）→スワンS2着（G2 京都1400m 12人気）→愛知杯4着（G3 中京1400m）
- 展開適性: 先行〜好位差し。左回り実績あり（中京愛知杯）
- コメント: スワンS2着はG2での最高重賞実績。知見「重賞実績（G3以上掲示板）はコース実績より優先」に合致。55kg軽量は牝馬の恩恵。東京コース実績はないが左回り中京での実績で補完。距離適性は1400mが最も合っていると判断。

### 8番 ララマセラシオン
- 評価: B
- 血統: カリフォルニアクローム × クロフネ（1200〜1400mスピード型）
- 近走: 2勝クラス1着→山城S1着（1400m）→阪急杯2着（G3 1400m 14人気）→高松宮記念16着（G1 1200m）
- 展開適性: 中位から差す
- コメント: 阪急杯2着は大穴での好走。14人気で1400m G3に対応できたことは実力の証明。高松宮記念16着は1200mが短すぎた可能性。1400mに戻れば再浮上のチャンス。東京コースは初だが距離適性を重視。

### 12番 ウイントワイライト
- 評価: B
- 血統: レイデオロ × ダイワメジャー（マイル系、1400m適性○）
- 近走: 精進湖特別1着（東京1400m稍重）→六甲アイラS12着（阪神1400m 重馬場）→節分S1着（東京1400m）→愛知杯7着（中京1400m）
- 展開適性: 先行〜好位
- コメント: 東京1400m2勝は圧倒的なコース適性の証明。六甲アイラSの12着は重馬場特異性の可能性。愛知杯は7着でG3では格落ちの感もあるが、東京に戻れば変わり目あり。4歳牝馬で55kg。知見「重賞で3〜5着が続く馬は△として残す」に準じて△に。

### 2番 ダノンセンチュリー
- 評価: B
- 血統: フィエールマン × Lope de Vega（父は長距離系だが母父マイル系で1400mは対応可能か）
- 近走: 五頭連峰特別2着（1600m）→1勝クラス1着（東京1600m 1人気）→2勝クラス1着（東京1600m 1人気）→雲雀S1着（東京1600m 2人気）
- 展開適性: 差し〜追込
- コメント: 東京1600m3連勝は東京コースへの高い親和性を示す。1400mへの距離短縮はやや不安だが、東京コース適性でカバーできるか。レーン騎手での重賞初挑戦。フィエールマン産駒が1400mでどこまで対応できるかが鍵。穴として△評価。

### 18番 ヤブサメ
- 評価: C
- 血統: ファインニードル × グラスワンダー（スプリンター系の父、1200m中心）
- 近走: 東山S1着（京都1400m 2人気）→タンザナイト3着（1200m）→淀短距離S1着（1200m 1人気）→シルクロードS5着（1200m G3）
- 展開適性: 逃げ〜先行
- コメント: 武豊騎乗で一発への期待。東山S1着（1400m）はあるが主戦場は1200m。外18番枠は多頭数で前半に位置取りが難しい可能性。ただし武豊の技術でカバーできるかも。スプリント系で1400mは少し長い血統的懸念あり。☆の低額押さえ。

### その他の馬
- **3番 カンチェンジュンガ**: セントウルS(G2)1着の実績はあるが1200m専門。近走阪急杯15着など惨敗続き。評価D
- **16番 ワールズエンド**: スワンS1人気8着が大きなマイナス。リゲルS2着等あるが1400mでは信頼できない。評価C
- **13番 ファンダム**: サートゥルナーリア産駒で中距離向き。オーシャンS2人気12着と1200mは短すぎ。ルメール騎乗で人気するが1400mへの適性疑問。評価C
- **11番 アサカラキング**: 1400m適性はあるが6歳で安定感欠く。評価C

## 注意事項
- ◎マイネルチケットの阪急杯凡走があるため、馬場状態（稍重以上）での変化には注意
- 18頭多頭数で外枠不利のため、枠順の影響が大きい（14番は7枠で中外枠）
- 当日の馬場状態が良馬場の場合、差し有利傾向が強まり評価が変わる可能性あり
- 知見「多頭数GII（15頭以上）は2頭軸フォーメーション」適用済み
- 知見「三連複軸流しの相手は評価C以上を広く」に従い2番・18番も相手に含めた
